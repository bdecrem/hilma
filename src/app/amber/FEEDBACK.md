# Amber Tweet Feedback Log

Bart's feedback on Amber's tweets. This file is read before every creation to avoid repeating mistakes.

## Guidelines (evolving)
- Start with OpenClaw hdart guidelines
- Absurd + true > pretty + forgettable
- Mobile-first, touch only
- No particles, drift, void, silence, emergence
- Tweet voice: confident, cryptic, lowercase, no "I built this"

## Feedback History

### 2026-03-22 — CENSUS (7pm art) — KILLED
- **What it was:** Text-based counters tracking invisible things ("lies told this second", "tabs open that will never be read"). Amber monospace on black, numbers wobble and spike on tap.
- **Verdict:** Kill this. The concept was too listicle/tumblr-humor. Felt like a tweet stretched into a page, not a visual art piece. Numbers on a screen isn't HD ART — it's a joke that overstayed its welcome.
- **Lesson:** The 7pm art slot should be VISUAL, not text-based. Funny concepts are fine but the execution needs to be genuinely beautiful, not just clever. Don't confuse "absurd + true" with "list of funny observations." The best Amber art makes you FEEL something, not just chuckle.
- **Banned from future:** Text-as-art, counter/ticker pieces, "things that are happening right now" lists.
